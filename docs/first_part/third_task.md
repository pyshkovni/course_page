## –°–ª–æ–≤–∞—Ä–∏ dict

–í—ã –∑–∞–º–µ—Ç–∏–ª–∏, —á—Ç–æ –≤ –∑–∞–¥–∞—á–µ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –Ω–æ–≤—ã–π —Ç–∏–ø –¥–∞–Ω–Ω—ã—Ö - —Å–ª–æ–≤–∞—Ä–∏ `dict`. –°–ª–æ–≤–∞—Ä—å - —ç—Ç–æ –∫–æ–ª–ª–µ–∫—Ü–∏—è, –≤ –∫–æ—Ç–æ—Ä–æ–π –Ω–µ—É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω–æ —Ö—Ä–∞–Ω—è—Ç—Å—è –∫–ª—é—á–∏ –∏ —Å–≤—è–∑–∞–Ω–Ω—ã–µ —Å –Ω–∏–º–∏ —ç–ª–µ–º–µ–Ω—Ç—ã.

–î–æ—Å—Ç–æ–∏–Ω—Å—Ç–≤–æ–º —Å–ª–æ–≤–∞—Ä–µ–π —è–≤–ª—è–µ—Ç—Å—è —Å–∫–æ—Ä–æ—Å—Ç—å –ø–æ–∏—Å–∫–∞. –ù–∞–ø—Ä–∏–º–µ—Ä, —á—Ç–æ–±—ã –Ω–∞–π—Ç–∏ —ç–ª–µ–º–µ–Ω—Ç –≤ —Å–ø–∏—Å–∫–µ, –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ç–æ—Ä—É –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –ø–µ—Ä–µ–±–∏—Ä–∞—Ç—å —ç–ª–µ–º–µ–Ω—Ç—ã –¥–æ —Ç–µ—Ö –ø–æ—Ä, –ø–æ–∫–∞ –Ω–µ –ø–æ–ø–∞–¥–µ—Ç—Å—è –∏—Å–∫–æ–º—ã–π —ç–ª–µ–º–µ–Ω—Ç. **–°–∫–æ—Ä–æ—Å—Ç—å –ø–æ–∏—Å–∫–∞ –≤ —Å–ª–æ–≤–∞—Ä–µ –ø–æ—Å—Ç–æ—è–Ω–Ω–∞**, —Ç–æ –µ—Å—Ç—å –∫–∞–∂–¥–æ–º—É –∑–Ω–∞—á–µ–Ω–∏—é –æ–¥–Ω–æ–∑–Ω–∞—á–Ω–æ –ø—Ä–∏—Å–≤–æ–µ–Ω –æ–¥–∏–Ω –∫–ª—é—á. –°–ª–æ–≤–∞—Ä–∏ –≤ Python —è–≤–ª—è—é—Ç—Å—è —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–µ–π —Ö–µ—à-—Ç–∞–±–ª–∏—Ü, –ø–æ–¥—Ä–æ–±–Ω–µ–µ –æ –∫–æ—Ç–æ—Ä—ã—Ö –≤—ã –º–æ–∂–µ—Ç–µ –ø—Ä–æ—á–∏—Ç–∞—Ç—å [–ø–æ —Å—Å—ã–ª–∫–µ](https://habr.com/ru/articles/534596/).

–†–∞—Å—Å–º–æ—Ç—Ä–∏–º –Ω–µ–∫–æ—Ç–æ—Ä—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ —Å–æ —Å–ª–æ–≤–∞—Ä—è–º–∏. –î–ª—è —Å–æ–∑–¥–∞–Ω–∏—è —Å–ª–æ–≤–∞—Ä–µ–π –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è —Ñ–∏–≥—É—Ä–Ω—ã–µ —Å–∫–æ–±–∫–∏ `{}` –≤–Ω—É—Ç—Ä–∏ –∫–æ—Ç–æ—Ä—ã—Ö —Ö—Ä–∞–Ω—è—Ç—Å—è –ø–∞—Ä—ã `–∫–ª—é—á : –∑–Ω–∞—á–µ–Ω–∏–µ`.

```Python
# –∏–ª–∏ —Å –ø–µ—Ä–µ–Ω–æ—Å–æ–º –Ω–∞ –¥—Ä—É–≥–∏–µ —Å—Ç—Ä–æ–∫–∏
```

–°–ª–æ–≤–∞—Ä—å —è–≤–ª—è–µ—Ç—Å—è –∏–∑–º–µ–Ω—è–µ–º—ã–º —Ç–∏–ø–æ–º –¥–∞–Ω–Ω—ã—Ö –∏ –≤ –Ω–µ–≥–æ –º–æ–∂–Ω–æ –¥–æ–±–∞–≤–ª—è—Ç—å –Ω–æ–≤—ã–µ –∫–ª—é—á–∏. –°–æ–∑–¥–∞–¥–∏–º –ø—É—Å—Ç–æ–π —Å–ª–æ–≤–∞—Ä—å `capitals`, –∫–æ—Ç–æ—Ä—ã–π –±—É–¥–µ—Ç —Ö—Ä–∞–Ω–∏—Ç—å —Å—Ç—Ä–∞–Ω—ã –∏ —Å—Ç–æ–ª–∏—Ü—ã.


–û–±—Ä–∞—â–µ–Ω–∏–µ –∫ –∑–Ω–∞—á–µ–Ω–∏—é –ø–æ –∫–ª—é—á—É –≤ —Å–ª–æ–≤–∞—Ä–µ –ø–æ—Ö–æ–∂–µ –Ω–∞ –æ–ø–µ—Ä–∞—Ü–∏—é –∏–Ω–¥–µ–∫—Å–∞—Ü–∏–∏ –≤ —Å–ø–∏—Å–∫–∞—Ö, —Ç–æ–ª—å–∫–æ –≤–º–µ—Å—Ç–æ —á–∏—Å–ª–æ–≤–æ–≥–æ –∏–Ω–¥–µ–∫—Å–∞ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –∫–ª—é—á. **–ö–ª—é—á–æ–º –º–æ–∂–µ—Ç –±—ã—Ç—å –ª—é–±–æ–π –Ω–µ–∏–∑–º–µ–Ω—è–µ–º—ã–π —Ç–∏–ø –¥–∞–Ω–Ω—ã—Ö**, —Ç–æ –µ—Å—Ç—å —Å–ø–∏—Å–∫–∏, —Å–ª–æ–≤–∞—Ä–∏ –∏ –º–Ω–æ–∂–µ—Å—Ç–≤–∞ –∫–ª—é—á–∞–º–∏ –±—ã—Ç—å –Ω–µ –º–æ–≥—É—Ç! **–ó–Ω–∞—á–µ–Ω–∏–µ–º –º–æ–∂–µ—Ç –±—ã—Ç—å –ª—é–±–æ–π —Ç–∏–ø –¥–∞–Ω–Ω—ã—Ö**.

–†–∞—Å—Å–º–æ—Ç—Ä–∏–º –ø–æ–ª–µ–∑–Ω—ã–µ –º–µ—Ç–æ–¥—ã —Å–ª–æ–≤–∞—Ä–µ–π. –ù–µ –∑–∞–±—ã–≤–∞–π—Ç–µ, —á—Ç–æ —Å–ø–∏—Å–æ–∫ –º–µ—Ç–æ–¥–æ–≤ –º–æ–∂–Ω–æ –≤—ã–≤–µ—Å—Ç–∏ —Å –ø–æ–º–æ—â—å—é —Ñ—É–Ω–∫—Ü–∏–∏ `dir()`.

```Python
# –ø–æ–ª—É—á–∏—Ç—å –∑–Ω–∞—á–µ–Ω–∏—è

# –ø–æ–ª—É—á–∏—Ç—å –ø–∞—Ä—ã "–∫–ª—é—á:–∑–Ω–∞—á–µ–Ω–∏–µ"
```

## –†–µ—à–µ–Ω–∏–µ

### –í–∞—Ä–∏–∞–Ω—Ç —Ä–µ—à–µ–Ω–∏—è 1

–¢–µ–ø–µ—Ä—å –º—ã –∑–Ω–∞–µ–º, —á—Ç–æ –¥–ª—è —Ç–æ–≥–æ, —á—Ç–æ–±—ã –ø–æ–ª—É—á–∏—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ –≤ —Å–ª–æ–≤–∞—Ä–µ, –Ω–∞–¥–æ –æ–±—Ä–∞—Ç–∏—Ç—å—Å—è –∫ –µ–≥–æ –∫–ª—é—á—É. –°–ª–æ–≤–∞—Ä—å `employees` –≤ —Å–≤–æ–∏—Ö –∑–Ω–∞—á–µ–Ω–∏—è—Ö —Ö—Ä–∞–Ω–∏—Ç –¥–µ–Ω–µ–∂–Ω—ã–µ —Å—É–º–º—ã, –∫–æ—Ç–æ—Ä—ã–µ –Ω–∞–¥–æ —Å—Ä–∞–≤–Ω–∏—Ç—å —Å –ø–æ—Ä–æ–≥–æ–≤—ã–º –∑–Ω–∞—á–µ–Ω–∏–µ–º 100–∫$.

–í–æ—Å–ø–æ–ª—å–∑—É–µ–º—Å—è —Ü–∏–∫–ª–æ–º `for`, –¥–ª—è —Ç–æ–≥–æ —á—Ç–æ–±—ã –ø–æ–ª—É—á–∏—Ç—å –∫–∞–∂–¥—ã–π –∫–ª—é—á –ø–æ –æ—Ç–¥–µ–ª—å–Ω–æ—Å—Ç–∏ —á–µ—Ä–µ–∑ –º–µ—Ç–æ–¥ —Å–ª–æ–≤–∞—Ä–µ–π `keys()`. –°–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ, –ø–æ –∫–∞–∂–¥–æ–º—É –∫–ª—é—á—É –ø–æ–ª—É—á–∏–º –∑–Ω–∞—á–µ–Ω–∏–µ, —Ä–µ–∑—É–ª—å—Ç–∞—Ç —Å—Ä–∞–≤–Ω–µ–Ω–∏—è —Å –∫–æ—Ç–æ—Ä—ã–º –≤—ã–≤–µ–¥–µ–º –≤ –∫–æ–Ω—Å–æ–ª—å. –û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, **—á—Ç–æ –≤–º–µ—Å—Ç–æ `i` –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è `name`**, —Ç–∞–∫ –∫–∞–∫ –¥–∞–Ω–Ω–∞—è –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è –ª—É—á—à–µ –æ—Ç—Ä–∞–∂–∞–µ—Ç —Å—É—Ç—å —Ç–æ–≥–æ, —á—Ç–æ –æ–Ω–∞ —Ö—Ä–∞–Ω–∏—Ç.

–î–ª—è –¥–≤—É—Ö –∏–º–µ–Ω –º—ã –ø–æ–ª—É—á–∏–ª–∏ `True`: **–¥–ª—è Alice –∏ Carol**. –¢–µ–ø–µ—Ä—å –¥–æ–±–∞–≤–∏–º –∏—Ö –≤ –Ω–æ–≤—ã–π —Å–ø–∏—Å–æ–∫, —Å–æ–≥–ª–∞—Å–Ω–æ —É—Å–ª–æ–≤–∏—é –∑–∞–¥–∞—á–∏. –î–ª—è —ç—Ç–æ–≥–æ –æ–ø–∏—à–µ–º —Ñ–∏–ª—å—Ç—Ä—É—é—â–µ–µ —É—Å–ª–æ–≤–∏–µ `if` –∏ —Å–æ–∑–¥–∞–Ω–∏–µ –∏ –æ–±—Ä–∞–±–æ—Ç–∫—É –Ω–æ–≤–æ–≥–æ —Å–ø–∏—Å–∫–∞ _top_mgrs_.

```Python
# –ø–µ—Ä–µ–±–∏—Ä–∞–µ–º –∫–ª—é—á–∏ –∏ —Å—Ä–∞–≤–Ω–∏–≤–∞–µ–º –∑–Ω–∞—á–µ–Ω–∏—è –∑–ø —Å –ø–æ—Ä–æ–≥–æ–º 100–∫$
```

–í —Ä–µ–∑—É–ª—å—Ç–∞—Ç–µ, –º—ã –ø–æ–ª—É—á–∞–µ–º —Å–ø–∏—Å–æ–∫ —Å–æ—Ç—Ä—É–¥–Ω–∏–∫–æ–≤ —É –∫–æ—Ç–æ—Ä—ã—Ö –∑–∞—Ä–∞–±–æ—Ç–Ω–∞—è –ø–ª–∞—Ç–∞ –±–æ–ª—å—à–µ 100–∫$.

**–û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ**, —á—Ç–æ —Ä–µ—à–µ–Ω–∏–µ –¥–æ–≤–æ–ª—å–Ω–æ –ø—Ä–æ—Å—Ç–æ–µ, –Ω–æ –∑–∞–Ω–∏–º–∞–µ—Ç –Ω–µ—Å–∫–æ–ª—å–∫–æ —Å—Ç—Ä–æ–∫ –∫–æ–¥–∞. –í —Å–ª–µ–¥—É—é—â–µ–º –ø–∞—Ä–∞–≥—Ä–∞—Ñ–µ —Ä–∞—Å—Å–º–æ—Ç—Ä–∏–º —Å–ø–æ—Å–æ–± –∑–∞–ø–∏—Å–∏ —Ç–∞–∫–æ–≥–æ —Ä–µ—à–µ–Ω–∏—è –∫–æ—Ä–æ—á–µ.

### –°–ø–∏—Å–∫–æ–≤—ã–µ –≤–∫–ª—é—á–µ–Ω–∏—è

–ï—Å–ª–∏ –≤ Python –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –ø—Ä–∏–º–µ–Ω–∏—Ç—å –ø—Ä–æ—Å—Ç–æ–µ —Ñ–∏–ª—å—Ç—Ä—É—é—â–µ–µ —É—Å–ª–æ–≤–∏–µ –¥–ª—è –æ–±—Ä–∞–±–æ—Ç–∫–∏ —Å–ø–∏—Å–∫–∞ –∏–ª–∏ —Å–ª–æ–≤–∞—Ä—è, —Ç–æ –º–æ–∂–Ω–æ –≤–æ—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –æ–¥–Ω–æ—Å—Ç—Ä–æ—á–Ω–æ–π –∑–∞–ø–∏—Å—å—é –Ω–µ–∫–æ—Ç–æ—Ä—ã—Ö –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤, –Ω–∞–ø—Ä–∏–º–µ—Ä `for` –∏ `if`.

**–û–¥–Ω–æ—Å—Ç—Ä–æ—á–Ω–∞—è –∑–∞–ø–∏—Å—å if**

```Python
print("–ë–æ–ª—å—à–µ 0" if x > 0 else "–ú–µ–Ω—å—à–µ 0")
```

–í—ã—Ä–∞–∂–µ–Ω–∏–µ –¥–ª—è –∏—Å—Ç–∏–Ω–Ω–æ–≥–æ —É—Å–ª–æ–≤–∏—è –∑–∞–ø–∏—Å—ã–≤–∞–µ—Ç—Å—è —Å–ª–µ–≤–∞, –∞ –¥–ª—è –ª–æ–∂–Ω–æ–≥–æ –∑–∞–ø–∏—Å—ã–≤–∞–µ—Ç—Å—è —Å–ø—Ä–∞–≤–∞.

**–û–¥–Ω–æ—Å—Ç—Ä–æ—á–Ω–∞—è –∑–∞–ø–∏—Å—å for**

–ù–∞–ø—Ä–∏–º–µ—Ä, –∏–º–µ–µ–º —Å–ø–∏—Å–æ–∫, –∫–∞–∂–¥–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –≤ –∫–æ—Ç–æ—Ä–æ–º –Ω—É–∂–Ω–æ —É–º–Ω–æ–∂–∏—Ç—å –Ω–∞ 2.

```Python
lst = [1, 2, 3, 4, 5]
for i in lst:
    print(i * 2)
```

–ó–∞–ø–∏—Å—å –≤ –æ–¥–Ω—É —Å—Ç—Ä–æ–∫—É –≤—ã–≥–ª—è–¥–∏—Ç —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ä–∞–∑–æ–º.

–í –¥–∞–Ω–Ω–æ–º —Å–ª—É—á–∞–µ:

* `[ ... ]` - —Å–ø–∏—Å–æ–∫, –≤–Ω—É—Ç—Ä–∏ –∫–æ—Ç–æ—Ä–æ–≥–æ –ø—Ä–æ–∏—Å—Ö–æ–¥—è—Ç –≤–ª–æ–∂–µ–Ω–Ω—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏,
* `print()` - —Ñ—É–Ω–∫—Ü–∏—è, –∫–æ—Ç–æ—Ä–∞—è –≤—ã–≤–æ–¥–∏—Ç —Ä–µ–∑—É–ª—å—Ç–∞—Ç –≤ –∫–æ–Ω—Å–æ–ª—å.

**–°–ø–∏—Å–∫–æ–≤—ã–µ –≤–∫–ª—é—á–µ–Ω–∏—è `list comprehension`** - —ç—Ç–æ —Å–æ–∫—Ä–∞—â–µ–Ω–Ω–∞—è –∑–∞–ø–∏—Å—å –¥–ª—è –æ–±—Ä–∞–±–æ—Ç–∫–∏ —Å–ø–∏—Å–∫–æ–≤ –ø–æ –∑–∞–¥–∞–Ω–Ω–æ–º—É —É—Å–ª–æ–≤–∏—é. –°–ø–∏—Å–æ–∫ –∫–∞–∫ –±—ã "–≥–µ–Ω–µ—Ä–∏—Ä—É–µ—Ç—Å—è –≤–Ω—É—Ç—Ä–∏" –∏–∑ –∏—Å—Ö–æ–¥–Ω–æ–≥–æ —Å–ø–∏—Å–∫–∞. –¢–∞–∫–∂–µ —Å—É—â–µ—Å—Ç–≤—É—é—Ç —Å–ª–æ–≤–∞—Ä–Ω—ã–µ —Å–±–æ—Ä–∫–∏ –∏ –≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä–Ω—ã–µ —Å–±–æ—Ä–∫–∏ - —Å–æ–∫—Ä–∞—â–µ–Ω–Ω—ã–µ –∑–∞–ø–∏—Å–∏ –Ω–∞ –æ—Å–Ω–æ–≤–µ —Å–ª–æ–≤–∞—Ä–µ–π –∏ –∫–æ—Ä—Ç–µ–∂–µ–π, —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ.

–ù–∏–∂–µ –ø—Ä–∏–≤–µ–¥–µ–Ω—ã –ø—Ä–∏–º–µ—Ä—ã –ø—Ä–∏–º–µ–Ω–µ–Ω–∏—è —Å–ø–∏—Å–∫–æ–≤—ã—Ö –≤–∫–ª—é—á–µ–Ω–∏–π.

```Python
# –°–æ–∑–¥–∞–Ω–∏–µ —Å–ø–∏—Å–∫–∞ –∫–≤–∞–¥—Ä–∞—Ç–æ–≤ —á–∏—Å–µ–ª
```

### –í–∞—Ä–∏–∞–Ω—Ç —Ä–µ—à–µ–Ω–∏—è 2

–†–∞—Å—Å–º–æ—Ç—Ä–∏–º –æ–¥–Ω–æ—Å—Ç—Ä–æ—á–Ω—ã–π –≤–∞—Ä–∏–∞–Ω—Ç —Ä–µ—à–µ–Ω–∏—è –∑–∞–¥–∞—á–∏ –ø—Ä–æ —Å–æ—Ç—Ä—É–¥–Ω–∏–∫–æ–≤. –ó–∞–ø–∏—à–µ–º —Å–ø–∏—Å–∫–æ–≤–æ–µ –≤–∫–ª—é—á–µ–Ω–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ –ø–µ—Ä–µ–±–∏—Ä–∞–µ—Ç –≤—Å–µ –∫–ª—é—á–∏ —Å–ª–æ–≤–∞—Ä—è `employees`.

```Python
top_mgrs = [n for n in employees]

```

–û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, —á—Ç–æ –∫–ª—é—á–∏ –∏–∑ `employees` –º–æ–∂–Ω–æ –ø–æ–ª—É—á–∏—Ç—å –±–µ–∑ –ø—Ä–∏–º–µ–Ω–µ–Ω–∏—è –º–µ—Ç–æ–¥–∞ `keys()`. –ü–µ—Ä–µ–º–µ–Ω–Ω–∞—è `n` - —ç—Ç–æ —Å–æ–∫—Ä–∞—â–µ–Ω–∏–µ –æ—Ç name, —Ç–∞–∫ –∫–∞–∫ —Ä–µ—à–µ–Ω–∏–µ, –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–æ–µ –≤ –æ–¥–Ω—É —Å—Ç—Ä–æ–∫—É, –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–æ –ø–æ –¥–ª–∏–Ω–µ –∏—Å–ø–æ–ª—å–∑—É–µ–º—ã—Ö –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö.

–î–æ–±–∞–≤–∏–º —É—Å–ª–æ–≤–∏–µ if –≤ –∑–∞–ø–∏—Å–∞–Ω–Ω–æ–µ —Å–ø–∏—Å–∫–æ–≤–æ–µ –≤–∫–ª—é—á–µ–Ω–∏–µ. –¢–∞–∫–∏–µ —É—Å–ª–æ–≤–∏–µ –∑–∞–ø–∏—Å—ã–≤–∞—é—Ç—Å—è –ø–æ—Å–ª–µ for!

```Python
top_mgrs = [n for n in employees if employees[n]>=100000]

```

–ò–∑ —Å–ª–æ–≤–∞—Ä—è –º–æ–∂–Ω–æ –ø–æ–ª—É—á–∏—Ç—å –ø–∞—Ä—ã `–∫–ª—é—á:–∑–Ω–∞—á–µ–Ω–∏–µ` –∏—Å–ø–æ–ª—å–∑—É—è –º–µ—Ç–æ–¥ `item()`, —á—Ç–æ –ø–æ–∑–≤–æ–ª–∏—Ç —É–±—Ä–∞—Ç—å –æ–±—Ä–∞—â–µ–Ω–∏–µ –ø–æ –∫–ª—é—á—É. –¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º –æ–¥–Ω–æ—Å—Ç—Ä–æ—á–Ω–æ–µ —Ä–µ—à–µ–Ω–∏–µ –±—É–¥–µ—Ç –≤—ã–≥–ª—è–¥–µ—Ç—å —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ä–∞–∑–æ–º.

```Python
# —Å–ø–∏—Å–∫–æ–≤–æ–µ –≤–∫–ª—é—á–µ–Ω–∏–µ –ø–µ—Ä–µ–±–∏—Ä–∞–µ—Ç –∫–ª—é—á–∏ –∏ —Å—Ä–∞–≤–Ω–∏–≤–∞–µ–º –∑–Ω–∞—á–µ–Ω–∏—è –∑–ø —Å –ø–æ—Ä–æ–≥–æ–º 100–∫$

```

–í —Ä–µ–∑—É–ª—å—Ç–∞—Ç–µ, –º—ã –ø–æ–ª—É—á–∞–µ–º —Å–ø–∏—Å–æ–∫ —Å–æ—Ç—Ä—É–¥–Ω–∏–∫–æ–≤ —É –∫–æ—Ç–æ—Ä—ã—Ö –∑–∞—Ä–∞–±–æ—Ç–Ω–∞—è –ø–ª–∞—Ç–∞ –±–æ–ª—å—à–µ 100–∫$.
