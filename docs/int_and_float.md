## –ú–∞—Ç–µ–º–∞—Ç–∏–∫–∞ –≤ Python

### –ê—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏

–†–∞–∑–±–µ—Ä–µ–º –ø–µ—Ä–≤—É—é –ø—Ä–æ—Å—Ç—É—é –∫–æ–º–∞–Ω–¥—É. –í—ã–≤–µ–¥–µ–º –Ω–∞ —ç–∫—Ä–∞–Ω —Å—É–º–º—É –¥–≤—É—Ö —á–∏—Å–µ–ª.


–í –æ—Ç–ª–∏—á–∏–∏ –æ—Ç –º–Ω–æ–≥–∏—Ö –¥—Ä—É–≥–∏—Ö —è–∑—ã–∫–æ–≤ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏—è –Ω–∞–º –Ω–µ –Ω—É–∂–Ω–æ –Ω–∏—á–µ–≥–æ –æ–ø—Ä–µ–¥–µ–ª—è—Ç—å –ø—Ä–µ–¥–≤–∞—Ä–∏—Ç–µ–ª—å–Ω–æ –ø–µ—Ä–µ–¥ —Å–æ–≤–µ—Ä—à–µ–Ω–∏–µ–º –æ–ø–µ—Ä–∞—Ü–∏–∏, —á—Ç–æ —É–ø—Ä–æ—â–∞–µ—Ç –∑–∞–ø–∏—Å—å –∫–æ–¥–∞.

* –û–±—ã—á–Ω–æ–µ –¥–µ–ª–µ–Ω–∏–µ `/`, —Ä–µ–∑—É–ª—å—Ç–∞—Ç–æ–º –∫–æ—Ç–æ—Ä–æ–≥–æ –±—É–¥–µ—Ç –¥—Ä–æ–±–Ω–æ–µ —á–∏—Å–ª–æ (–¥–∞–∂–µ –µ—Å–ª–∏ —á–∏—Å–ª–æ –¥–µ–ª–∏—Ç—Å—è –Ω–∞—Ü–µ–ª–æ).
